IRIDIUM

DISPOSITIVO DE MONITORAMENTO, COLETA E TRANSMISSAO DE DADOS DE VOO

A coleta e transmissão de dados de voo é feta por um moderno dispositivo eletrônico instalado de maniera simples, rápida, com baixo custo de aquisição e manutenção.
Recebe e transmite dados via rede de telefonia celular e quando ela não estiver disponível, de forma automatizada por meio da Rede de Satélites Iridium.

TMF-01

UNIDADE REMOTA DE MONITORAMENTO E CONTROLE DE ABASTECIMENTO DE COMBUSTÍVEL

Fabricado de forma robusta para operar em caminhões tanques ou unidades fixas. Possui alimentação de 24VDC, duas entradas para medidores de fluxo pulsantes (Primário e Secundário) e uma interface RS-232 para modens GPRS, Satelital ou aplicações IoT.
Este sistema não possui preço fixo. O mesmo varia de acordo com as horas voadas e com a quantidade de módulos a serem adquiridos.
A partir de fevereiro de 2020 o Followair passou a operar com os dados e informações provenientes e disponibilizados pelo ADB B (Automatic Dependent Surveillance-broadcast) Caso o cliente decida em não adquirir o rastreador, possuímos API com o Flight Cell e já disponível com o Skytrack para buscar as posições.
